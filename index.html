CI/CD Github Actions Workshop

Pre-requisite:

Clone Repository [healthcare-gitub-actions-workshop] 
								
									$ git clone git@github.com:healthcarecom/healthcare-gitub-actions-workshop.git
Create a new branch 
								
									$ git checkout -b *apinzon*

Follow this format to name your new branch, Use the first character of your first name followed by your complete last name example: to create a new branch for user Alejandro pinzon, it would be apinzon, add it to the word cicd.

Create Your First Github Actions Workflow

Create a Workflow File

  1. As a part of this branch, create a workflows directory nested inside the .github directory.
  2. In the new .github/workflows/ directory, create a file titled main.yml

Add the following content to the main.yml file

						
							name: My First CI/CD Pipeline with Github Actions
							on:
							push:
								branches: [cicd-apinzon]
